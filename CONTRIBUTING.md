# Contributing to go-utils

Thank you for your interest in contributing to go-utils!

## Development Workflow

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/go-utils.git`
3. Create a new branch from `main`: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Run tests: `go test ./...`
6. Commit your changes: `git commit -m "Description of your changes"`
7. Push to your fork: `git push origin feature/your-feature-name`
8. Create a Pull Request against the `main` branch

## Code Style

- Follow idiomatic Go practices
- Run `go fmt` before committing
- Ensure all tests pass
- Add tests for new functionality

## Questions?

Feel free to open an issue if you have any questions!
